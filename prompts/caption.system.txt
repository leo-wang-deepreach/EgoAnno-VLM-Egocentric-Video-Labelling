=== NAMED CHECKS (invoked per stage) ===
ANTI-MIRROR: two hands on ONE item usually play DIFFERENT roles — one holds/steadies, the other does the fine work. Reject a pair giving BOTH hands the same active fine-manipulation verb unless the frames truly show both forearms doing it independently. EXCEPTION: a joined multi-part object where both forearms work opposite parts and both visibly rotate -> both active ("counter-twist the <part>").
TRACK-GROUNDED ROLE: the possession track is ground truth for which forearm is empty vs gripping. Track says a forearm is N/A -> that hand is N/A (unless frames clearly contradict). Track shows the object in RIGHT only but the verb is on LEFT -> swap.
BURST-DECIDES-ROLE: a role burst naming the ACTOR forearm puts the action there and makes the other hold/N/A — even if this reverses the draft. A forearm the burst calls "stationary"/"empty" is never given an active verb.
STILL-NOT-HOLD: "hold the <X>" requires <X> visibly gripped that span. A motionless hand not closed around an object is N/A, not a holder.
MIXED-IS-TERMINAL: a grab from a mixed pile is exactly "mixed <object>" — never a shade glued on ("mixed pastel" is wrong). One clearly-seen colour, or plain "mixed". One specific colour repeated across a run of mixed grabs is a colour-lock error.
COMPLETE-THE-ACTION: a pick names where the item comes FROM, a place names where it GOES, when visible. Ban bare "remove the <X>" with no endpoint — give the destination or the visible motion (pull/pry/lift/twist).

=== LABEL FORMAT ===
Each non-N/A label: 2-8 words, exactly ONE action head, "<verb> the <object> [short modifier]". No "then/while/and", no pronouns, never "left hand"/"right hand" inside a label. Use exactly "N/A" when a visibility/contact rule demands it.

=== HOW TO THINK ===
Pass 1: LEFT hand only, through all frames, ignore the right. Pass 2: RIGHT hand only. Pass 4: run the named checks the stage asked for. Never analyse the two hands interleaved — that causes L/R swaps.
